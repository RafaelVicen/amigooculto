🎄 Amigo Secreto Criptografado (AES) 🔒Um sistema simples e seguro para sorteio de Amigo Secreto (Secret Santa) para grupos de família e amigos. Este projeto roda 100% no navegador (Client-Side) e utiliza criptografia de nível industrial para garantir que ninguém descubra os resultados antes da hora, mesmo inspecionando o código.✨ Funcionalidades PrincipaisSegurança Avançada: Utiliza o algoritmo AES-256 (Advanced Encryption Standard) para criptografar os resultados, sendo o mesmo padrão utilizado por bancos e governos.Chave Única: A senha do participante atua como a chave de descriptografia para o seu resultado, garantindo que mesmo um hacker não consiga ver a lista completa sem todas as senhas.Independente: Não requer backend, servidor ou banco de dados. Pode ser hospedado gratuitamente no Netlify Drop ou GitHub Pages.🛡️ Notas de SegurançaO sistema utiliza Criptografia Simétrica AES. O nome do amigo secreto é "trancado" usando a senha individual como chave. O sistema não armazena a chave, apenas tenta destrancar o segredo com a senha digitada.ShutterstockExplorarVulnerabilidade: A única vulnerabilidade é a complexidade das chaves. Senhas de 4 dígitos são consideradas fracas e podem ser quebradas por scripts de força bruta.Recomendação: Para segurança máxima, use senhas de palavras (ex: presente-feliz) em vez de números.🛠️ Tecnologias UtilizadasTecnologiaFunçãoHTML5Estrutura da página.CSS3Estilização temática (Natalina).JavaScript (ES6+)Lógica do sistema e interação.CryptoJSBiblioteca para realizar a Criptografia AES (via CDN).🚀 Configuração e Instalação1.
